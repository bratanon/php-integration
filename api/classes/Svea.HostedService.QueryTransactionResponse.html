\Svea\HostedService

\Svea\HostedServiceQueryTransactionResponse

QueryTransactionResponse handles the query transaction response

Note that the amount and vat attributes only reflect the initial payment transaction result, they are not authorative after i.e. loweramount has been called on a transaction, see the authorizedamount attribute.

There may be inconsistencies in how class attributes are capitalized, this is due to a wish to conform with pre-existing attribute names. Please refer to the phpdoc for an authoritative listing of all response attribute names.
