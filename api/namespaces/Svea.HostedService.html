Classes

AnnulTransaction AnnulTransaction is used to cancel (annul) a card transaction.
AnnulTransactionResponse AnnulTransactionResponse handles the annul transaction response
CardPayment Goes to PayPage and excludes all methods that are not card payments
ConfirmTransaction confirmTransaction can be performed on transactions having the status AUTHORIZED at Svea. After as successful request, the transaction will have status CONFIRMED and that will be captured on the given capturedate.
ConfirmTransactionResponse ConfirmTransactionResponse handles the confirm transaction response
CreditTransaction creditTransaction can be used to credit transactions. Only transactions that have reached the status SUCCESS at Svea can be credited.
CreditTransactionResponse CreditTransactionResponse handles the credit transaction response
DirectPayment Go to PayPage and exclude all methods that are not direct bank payments.
ExcludePayments
GetPaymentMethods Returns an array of SystemPaymentMethods with all paymentmethods conected to the merchantId and/or ClientId
HostedAdminResponse Handles diverse administrative function responses from the webservice and wrapped legacy services through the webservice.
HostedOrderRowBuilder

HostedXmlBuilder Formats request xml in preparation of sending request to hosted webservice.
ListPaymentMethods HostedRequest is the parent of hosted webservice requests.
ListPaymentMethodsResponse ListPaymentMethodsResponse handles the getpaymentmethods transaction response consistent with other HostedAdmin responses (i.e. returns an HostedResponse object)
LowerTransaction lowerTransaction modifies the amount in an existing card transaction having status AUTHORIZED or CONFIRMED. If the amount is lowered by an amount equal to the transaction authorized amount, then after a successful request the transaction will get the status ANNULLED.
LowerTransactionResponse LowerTransactionResponse handles the lower transaction response
PaymentForm Class used by HostedPayment Contains: Complete form without submit in html format: $completeHtmlFormWithSubmitButton Array of formfields in html format: $htmlFormFieldsAsArray Values for form: $merchantId, $xmlMessageBase64, $mac Array of formfields in raw format: $xmlMessage Raw fields: $xmlMessageBase64, $message, $merchantId, $secretWord, $mac
PaymentMethodPayment HostedPayment and its descendants sets up the various hosted payment methods.

QueryTransaction Query information about an existing card or direct bank transaction.
QueryTransactionResponse QueryTransactionResponse handles the query transaction response
RecurTransaction Performs a recurring card transaction, using a previously set up subscription id.
RecurTransactionResponse RecurTransactionResponse handles the recur transaction response
